# 故事设定

## 设定基调

- 共鸣价值观
    - 只想做事儿，会被背刺
- 共鸣场景
    - 被窃取项目成果
- 负面情绪 / 生活洞察
    - 背刺
- 人类弱点
    - 贪婪、自私
- 调动观众情绪
    - 委屈
- 主角
    - 雪花

## 设定主角，雪花

- 弱点 = 太相信做事本身 = 容易输给布局的人
    - 他以为把风车图纸画出来，大家自然会理解
    - 他以为农场变好了，大家自然会记得是谁做的
    - 他以为黑猪不干活，就不会有人支持黑猪
    - 他以为功劳和真相，是不需要经营的

## 设定配角，黑猪

- 只是阴险、贪婪、自私 = 脸谱化
- 方案 = 黑猪懂动物的恐惧
    - 雪花给动物讲未来
        - 风车建好后，大家会更轻松
    - 黑猪给动物讲风险
        - 风车会失败，粮食会不够，人类会打回来，雪花会把大家害死

> 现实里很多夺权者，不是靠比你更能干，而是靠制造恐惧、制造怀疑、制造敌人

## 打造共鸣点

- 雪花以为，只要自己把农场建设好，大家就会认可他
- 可他没想到，在职场里，最危险的不是敌人，而是那个一直躲在阴影里、等着摘果子的人

## 打造冲突 / 对立

- 具体是谁和谁有矛盾？
- 因为什么产生矛盾的？

## 夺权过程：怎么处理

- 转折不能太快
- 塑造阴谋感，暗流涌动的过程
    - 黑猪夺权，是蓄谋已久的
- 作用
    - 产生观众情绪
        - 观众能看见危险逼近
        - 但雪花看不到
- 步骤
    - Step 1：塑造雪花的实干
    - Step 2：黑猪经营实力，偷偷搞事情
        - 有布局 = 让夺权过程更合理，不突兀
    - Step 3：夺权
        - 先夺话语权，而不是先动手
        - 雪花让人心疼 = 不是输给能力，而是输给阴谋

## 雪花的下场：怎么处理

- 不能立马死
    - 无法塑造被窃取劳动果实的痛感
    - 无法塑造委屈感，不能带动情绪
- 不能生活在农场内？
    - 如果生活在农场内，解释不清为什么还有饭吃
- 选择流亡
- 流亡后，偶尔回来看一看
    - 立了牌子，写上是农场公敌
    - 写了雪花的“罪行”
    - 流亡后，很多农场事故，也写成了是雪花暗中捣乱
        - 粮仓失火
        - 庄稼减产
        - 动物失踪
- 雪花转身离开，什么也没有说，默默承受一切

> 你付出的一切，被别人夺走；你留下的成果，被别人占有；最后连你的名字都被抹掉

## 情绪钩子 = 让观众为雪花感到委屈

- 他明明是在为所有动物做事
- 他明明是最早冲在前面的人
- 他明明没有争权，只想把风车建起来
- 最后却被说成叛徒，连他的成果都成了别人的功劳

# 大纲目录

## 开头钩子

为什么有些人明明最努力，最后却最先出局？

因为他以为，只要把事做好，大家就会记得他。

可雪花不知道，有人从一开始，就不是来做事的，是来等着摘果子的。

## 动物们被农场主压迫（人类的自私）

雪花是一头白猪，生活在芬奇农场

农场里有各种动物，被农场主剥削

忍受着饥饿、寒冷、鞭打，睡在粪便里

农场主窃取动物们的劳动果实，用于自己的享乐

## 反抗人类农场主（为自由而奋斗）

农场主一天没出现了，动物们饿的受不了

奶牛冲开大门，其他动物们也一冲而上，到处寻找吃的

农场主和他的伙计们，拿着鞭子出来阻止

动物们不约而同得冲向人类，用尽全身的力气，冲撞、踢踹

雪花冲锋在最前面，硬扛着鞭子，撞伤弄农场主

人类好像从没见过动物疯狂的样子，竟然吓得惊慌失措，拔腿就跑

动物们竟然敢跑了人类，迎来了自由

```
它的作用不是讲革命，而是证明：雪花不是嘴上厉害，他是真的敢冲、敢扛事
```

## 自由后的混乱（动物的本性就是自私的）

第二天，雪花像往常一样起来，准备迎接自由的空气

失望的是，农场好像并没有变得更好

牛和马在抢最后一点草料

鸡和鸭在抢最后一点水喝

猫偷走了母鸡的鸡蛋

```
但重点不是“动物自私”，而是：

没有人类以后，动物们才发现，自由不是结局，混乱才刚开始。

这能给雪花一个站出来的理由。
```

## 组织动物建设农场，结束混乱（为美好生活而奋斗）

雪花心想，这样下去不行啊，没几天大家就都得饿死

就站出来，组织动物们协同劳作，恢复农场的正常运转

因为反抗人类时，雪花表现出来的勇猛，让动物们愿意听他的

虽然有个别动物偷懒、唱反调，但农场也逐渐好转起来

为了进一步让大家团结协作，雪花发布了规则，所有动物一律平等

粮食大丰收，动物们幸福感爆棚，一切似乎在向着好的方向发展

```
这里要写实干：

分粮食、排班、修仓库、组织耕地、处理争吵。

让观众看到：雪花不是抢权，他是在救场。
```

## 风车计划

```
这是核心段落。

雪花发现农场最大的问题：动物们每天累到半死，还是吃不饱。
于是他开始研究风车。

他用蹄子在地上画图。
晚上别的动物睡了，他还趴在谷仓里改图纸。
他想让风车磨谷、抽水、发电，让动物们不再像过去一样被活活榨干。

这里观众会开始喜欢他。
```

## 黑猪先夺话语权

```
这一段非常关键。

雪花开会讲风车。动物们听得眼睛发亮。
他刚讲完，黑猪站起来，不谈方案，只问几个问题：

风车失败了怎么办？
粮食不够了怎么办？
人类趁机打回来怎么办？
如果雪花错了，谁来负责？

这不是能力竞争，这是舆论战。

雪花解释技术，黑猪制造恐惧。
雪花讲未来，黑猪讲危险。
动物们开始动摇。
```

```
这一刻要突出：

雪花不是输给了更好的方案。
```

## 成果被窃取

```

但这不是雪花的计划。
这是黑猪早就想好的伟大决策。
```

## 流亡后的委屈

```

一个下雪的夜里，雪花偷偷回到山坡上。

因为农场里，已经没有动物愿意听他说话了。

这个结尾很有委屈感。
```
