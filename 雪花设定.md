
# 设定基调

- 共鸣价值观
    - 只想做事儿，会被背刺
- 共鸣场景
    - 被窃取项目成果
- 负面情绪 / 生活洞察
    - 背刺
- 人类弱点
    - 贪婪、自私
- 调动观众情绪
    - 委屈
- 主角
    - 雪花

# 设定主角，雪花

- 弱点 = 太相信做事本身 = 容易输给布局的人
    - 他以为把风车图纸画出来，大家自然会理解
    - 他以为农场变好了，大家自然会记得是谁做的
    - 他以为黑猪不干活，就不会有人支持黑猪
    - 他以为功劳和真相，是不需要经营的

# 设定配角，黑猪

- 只是阴险、贪婪、自私 = 脸谱化
- 方案 = 黑猪懂动物的恐惧
    - 雪花给动物讲未来
        - 风车建好后，大家会更轻松
    - 黑猪给动物讲风险
        - 风车会失败，粮食会不够，人类会打回来，雪花会把大家害死

> 现实里很多夺权者，不是靠比你更能干，而是靠制造恐惧、制造怀疑、制造敌人

# 打造共鸣点

- 雪花以为，只要自己把农场建设好，大家就会认可他
- 可他没想到，在职场里，最危险的不是敌人，而是那个一直躲在阴影里、等着摘果子的人

# 打造冲突 / 对立

- 具体是谁和谁有矛盾？
- 因为什么产生矛盾的？

# 夺权过程：怎么处理

- 转折不能太快
- 塑造阴谋感，暗流涌动的过程
    - 黑猪夺权，是蓄谋已久的
- 作用
    - 产生观众情绪
        - 观众能看见危险逼近
        - 但雪花看不到
- 步骤
    - Step 1：塑造雪花的实干
    - Step 2：黑猪经营实力，偷偷搞事情
        - 有布局 = 让夺权过程更合理，不突兀
    - Step 3：夺权
        - 先夺话语权，而不是先动手
        - 雪花让人心疼 = 不是输给能力，而是输给阴谋

# 雪花的下场：怎么处理

- 不能立马死
    - 无法塑造被窃取劳动果实的痛感
    - 无法塑造委屈感，不能带动情绪
- 不能生活在农场内？
    - 如果生活在农场内，解释不清为什么还有饭吃
- 选择流亡
- 流亡后，偶尔回来看一看
    - 立了牌子，写上是农场公敌
    - 写了雪花的“罪行”
    - 流亡后，很多农场事故，也写成了是雪花暗中捣乱
        - 粮仓失火
        - 庄稼减产
        - 动物失踪
- 雪花转身离开，什么也没有说，默默承受一切

> 你付出的一切，被别人夺走；你留下的成果，被别人占有；最后连你的名字都被抹掉

# 情绪钩子 = 让观众为雪花感到委屈

- 他明明是在为所有动物做事
- 他明明是最早冲在前面的人
- 他明明没有争权，只想把风车建起来
- 最后却被说成叛徒，连他的成果都成了别人的功劳
