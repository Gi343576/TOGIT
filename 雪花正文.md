
# 开头钩子

为什么有些人明明最努力，最后却最先出局？

因为他以为，只要把事做好，大家就会记得他。

可雪花不知道，有人从一开始，就不是来做事的，是来等着摘果子的。

# 动物们被农场主压迫（人类的自私）

雪花是一头白猪，生活在芬奇农场

农场里有各种动物，被农场主剥削

忍受着饥饿、寒冷、鞭打，睡在粪便里

农场主窃取动物们的劳动果实，用于自己的享乐

# 反抗人类农场主（为自由而奋斗）

农场主一天没出现了，动物们饿的受不了

奶牛冲开大门，其他动物们也一冲而上，到处寻找吃的

农场主和他的伙计们，拿着鞭子出来阻止

动物们不约而同得冲向人类，用尽全身的力气，冲撞、踢踹

雪花冲锋在最前面，硬扛着鞭子，撞伤弄农场主

人类好像从没见过动物疯狂的样子，竟然吓得惊慌失措，拔腿就跑

动物们竟然敢跑了人类，迎来了自由

```
它的作用不是讲革命，而是证明：雪花不是嘴上厉害，他是真的敢冲、敢扛事
```

# 自由后的混乱（动物的本性就是自私的）

第二天，雪花像往常一样起来，准备迎接自由的空气

失望的是，农场好像并没有变得更好

牛和马在抢最后一点草料

鸡和鸭在抢最后一点水喝

猫偷走了母鸡的鸡蛋

```
但重点不是“动物自私”，而是：

没有人类以后，动物们才发现，自由不是结局，混乱才刚开始。

这能给雪花一个站出来的理由。
```

# 组织动物建设农场，结束混乱（为美好生活而奋斗）

雪花心想，这样下去不行啊，没几天大家就都得饿死

就站出来，组织动物们协同劳作，恢复农场的正常运转

因为反抗人类时，雪花表现出来的勇猛，让动物们愿意听他的

虽然有个别动物偷懒、唱反调，但农场也逐渐好转起来

为了进一步让大家团结协作，雪花发布了规则，所有动物一律平等

粮食大丰收，动物们幸福感爆棚，一切似乎在向着好的方向发展

```
这里要写实干：

分粮食、排班、修仓库、组织耕地、处理争吵。

让观众看到：雪花不是抢权，他是在救场。
```

# 风车计划

```
这是核心段落。

雪花发现农场最大的问题：动物们每天累到半死，还是吃不饱。
于是他开始研究风车。

他用蹄子在地上画图。
晚上别的动物睡了，他还趴在谷仓里改图纸。
他想让风车磨谷、抽水、发电，让动物们不再像过去一样被活活榨干。

这里观众会开始喜欢他。

# 项目成果具体化

- 不能泛泛得说建设农场
- 要用能改变农场命运的大项目，证明雪花的价值
    - 风车 = 项目成果、职业理想、贡献筹码
- 风车 = 雪花被背刺、被窃取成果、被污名化的核心证据
```

# 黑猪开始暗中布局（平静的水面下，一定暗流涌动）

事情总不会一帆风顺。如果事情一帆风顺，那大概率就是暴风雨前的宁静

```
不要立刻放狗。

先写几组小动作：

雪花讲风车时，黑猪不反对，只是冷笑。
雪花画图时，黑猪偷偷把几只小狗带走。
雪花组织大家干活时，黑猪在角落里和羊群说话。
没过几天，羊群开始重复一句话：
“风车太危险，雪花不可靠。”

这就是阴谋感。
```

# 黑猪先夺话语权

```
这一段非常关键。

雪花开会讲风车。动物们听得眼睛发亮。
他刚讲完，黑猪站起来，不谈方案，只问几个问题：

风车失败了怎么办？
粮食不够了怎么办？
人类趁机打回来怎么办？
如果雪花错了，谁来负责？

这不是能力竞争，这是舆论战。

雪花解释技术，黑猪制造恐惧。
雪花讲未来，黑猪讲危险。
动物们开始动摇。
```

# 夺权高潮：放狗驱逐雪花

```
等雪花快要赢得支持时，黑猪才动手。

九条猎犬冲出来。
雪花这时才明白，黑猪不是临时翻脸，而是早就准备好了。

这一刻要突出：

雪花不是输给了更好的方案。
他是输给了早就埋好的刀。
```

# 成果被窃取

```
黑猪赶走雪花后，立刻宣布：

风车还是要建。
但这不是雪花的计划。
这是黑猪早就想好的伟大决策。
雪花只是偷走图纸、破坏农场、勾结人类的叛徒。

这才是最痛的地方。

不是你失败了。
是你成功了，但成功不属于你。
```

# 流亡后的委屈

```
你提出“雪花偶尔回来看看”，这个可以用，但不要写太多次，一次就够。

比如：

一个下雪的夜里，雪花偷偷回到山坡上。
他看见农场门口立了一块牌子。
牌子上写着：农场公敌——雪花。
下面列着他的罪行：偷粮、纵火、破坏风车、勾结人类。

而远处，那座风车正在一点点立起来。

雪花看了很久。
最后转身离开。
他没有解释。
因为农场里，已经没有动物愿意听他说话了。

这个结尾很有委屈感。
```
