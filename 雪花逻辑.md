# 打造共鸣点

- 雪花以为，只要自己把农场建设好，大家就会认可他
- 可他没想到，在职场里，最危险的不是敌人，而是那个一直躲在阴影里、等着摘果子的人

# 项目成果具体化

- 不能泛泛得说建设农场
- 要用能改变农场命运的大项目，证明雪花的价值
    - 风车 = 项目成果、职业理想、贡献筹码
- 风车 = 雪花被背刺、被窃取成果、被污名化的核心证据

# 塑造对立

- 不要写“黑猪上位”
- 要写黑猪“怎么上位”

# 转折预告

- 观众以为，故事圆满。实际上，最大的转折才刚要出现
- 删掉：“讲课”类文字
    - 是作者讲的，容易出戏
    - “就像辩证法说的，旧的主要矛盾解决了，就会产生新的矛盾”
- 删掉：“说得太满”的文字
    - 更像下结论
    - 不像讲故事
    - “如今驱赶了外敌，必然要陷入内斗”

# 夺权过程：怎么处理

- 转折不能太快
- 塑造阴谋感，暗流涌动的过程
    - 黑猪夺权，是蓄谋已久的
- 作用
    - 产生观众情绪
        - 观众能看见危险逼近
        - 但雪花看不到
- 方案 = 3 步
    - 塑造雪花的实干
    - 黑猪经营实力，偷偷搞事情
        - 有布局 = 让夺权过程更合理，不突兀
    - 夺权 = 先夺话语权，而不是先动手
        - 雪花让人心疼 = 不是输给能力，而是输给阴谋

# 雪花的下场：怎么处理

- 不能立马死
    - 无法塑造被窃取劳动果实的痛感
    - 无法塑造委屈感，不能带动情绪
- 流亡 / 生活在农场内？
    - 流亡
    - 如果生活在农场内，解释不清为什么还有饭吃
- 流亡后，偶尔回来看一看
    - 立了牌子，写上是农场公敌
    - 写了雪花的“罪行”
    - 流亡后，很多农场事故，也写成了是雪花暗中捣乱
        - 粮仓失火
        - 庄稼减产
        - 动物失踪
- 雪花转身离开，什么也没有说，默默承受一切

> 你付出的一切，被别人夺走；你留下的成果，被别人占有；最后连你的名字都被抹掉

# 情绪钩子 = 让观众为雪花感到委屈

- 他明明是在为所有动物做事
- 他明明是最早冲在前面的人
- 他明明没有争权，只想把风车建起来
- 最后却被说成叛徒，连他的成果都成了别人的功劳
